This is a java program and can be run on any java version above 1.6. 
You need to have java software installed.

Steps:

1. The java program needs an input directory for questions directory which is the set of java files.
2. The absolute path of the input directory has to be given for listFiles. 
3. It also needs an existing text file which is created and the path of text file has to given as input for method. 
4. The method executePrograms needs the absolute path to the input file which is already created in the local directory.
